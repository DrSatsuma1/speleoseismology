## Methodology

Anomalies classified BEFORE checking earthquake catalogs using:
1. **Coupling ratio**: |δ18O_z| / |δ13C_z| - values near 1.0 = SEISMIC (coupled), >3.0 = CLIMATIC (decoupled)
2. **δ13C absolute values**: > -9.0‰ = geogenic CO₂ (seismic indicator), < -10.5‰ = biogenic (climatic)
3. **Multi-year clusters**: Consecutive anomalies suggest real events
4. **Recovery pattern**: Sawtooth (gradual) = seismic, spike = climatic

**NOTE**: 402 CE event previously analyzed - excluded from blind test scoring but included for completeness.
