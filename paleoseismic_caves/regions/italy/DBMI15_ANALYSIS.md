## EXECUTIVE SUMMARY

The Italian Macroseismic Database (DBMI15 v4.0) provides 123,981 macroseismic data points (MDPs) from 3,229 earthquakes spanning 1005-2020 CE. This analysis cross-references the database with the Basura Cave speleothem anomalies to validate our "Dark Earthquake" hypothesis.

### Key Findings

| Finding | Significance |
|---------|--------------|
| **1394 ABSENT from DBMI15** | Confirms "Dark Earthquake" hypothesis - zero MDPs for 1394 |
| **1285 present but DISTANT** | December 13, 1285 in Ferrara (282 km east) - different fault system |
| **260-year Ligurian gap** | No Ligurian earthquakes recorded 1276-1536 CE |
| **1545 PRESENT** | Mystery anomaly now explained - two Imax 7-8 events |

## APPENDIX: RELATIONSHIP BETWEEN CPTI15 AND DBMI15

- **CPTI15**: Parametric catalog (magnitude, location parameters for each earthquake)
- **DBMI15**: Macroseismic database (individual intensity observations at specific places)

DBMI15 is the underlying observational data from which CPTI15 parameters are derived. An earthquake in CPTI15 has one entry; in DBMI15 it may have dozens or hundreds of MDPs.

**Example**: The 1348 Carinthian earthquake
- CPTI15: Single entry with Mw ~6.9
- DBMI15: 89 individual intensity observations across the region
